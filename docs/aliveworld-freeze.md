# AliveWorld Freeze

**Дата:** 2026-06-12

**Причина заморозки:**

Aliveworld начал генерировать события и слухи раньше, чем в мире появились реальные
поселения, дороги, жители и хозяйственные процессы. Это привело к необходимости
задним числом материализовывать объекты для уже созданных абстрактных событий.
Дороги приходилось прокладывать по абстрактным маршрутам, а сцены событий —
размещать поверх неструктурированного ландшафта.

Текущее направление — **PerfectWorld**: физическое устройство мира (регионы,
поселения, дороги, здания, фермы) создаётся до запуска симуляции событий.
Aliveworld будет разморожен после того, как PerfectWorld создаст достаточно
полноценный населённый мир.

## Текущее состояние AliveWorld

- Календарь, tick-система — работают
- Environment bridge — работает
- Поселения — логические (не физические)
- Мировые события, слухи, хроника — работают
- Сайты (abstract → anchored → materialized) — работают
- Маршруты (planned → materialized) — работают
- Route materialization (v2) — budgeted job, дорожное полотно
- GPS/радар HUD, трекинг — работают
- Remote controller — работает
- Terrain survey для anchoring — работает
- Budgeted job runner — работает

## Проблемы, приведшие к смене направления

1. **Абстрактные события предшествовали физическому миру.** События создавались
   на пустом ландшафте, что требовало дорогостоящей материализации задним числом.
2. **Поселения не были физическими.** NPC, здания, жители отсутствовали.
3. **Дороги не вели к реальным местам.** Маршруты прокладывались между
   абстрактными точками, а не между существующими поселениями.
4. **Событийные сцены обнажили архитектурный предел:** физические следы событий
   приходилось размещать у абстрактных sites/routes, а не у уже существующих
   дорог, хозяйств и населённых пунктов.

## Что должно появиться в PerfectWorld до разморозки AliveWorld

- [ ] Региональный планировщик (regions 1024×1024)
- [ ] Физические поселения (farm, hamlet, village)
- [ ] Дорожная сеть между поселениями
- [ ] Реестр строений и схем
- [ ] Материализация через mapgen (Lua mapgen)
- [ ] Базовые жители и хозяйство
- [ ] Торговые тракты
- [ ] Интеграционный слой: PerfectWorld → AliveWorld

## Как включить AliveWorld обратно

```bash
# 1. Раскомментировать load_mod_aliveworld_* в world.mt
# 2. Закомментировать load_mod_perfectworld = true
# 3. Перезапустить сервер
```
